How Does This Work?

Standing on the backs of giants :) There is an application called ImageMagick that allows you to do things like blend two images together into one image. My initial theory was: if I can just step through all of the image files in a folder, I can progressively combine each image with the image that came before it and create one "merged" version of them all at the end.

Unfortunately this idea didn't work so well. When you combine the first two images together, everything looks fine. When you combine the third image with the newly-created merged image of the first two images, they will get combined unequally. The third image takes up 50% and the first two images combined take up the other 50%. So as you go forward, each new image obliterates whatever came before it, and you end up with whatever the last image in the folder is, plus some blurriness :(

The solution is to do the combinations in phases. First, you go through the folder of images and combine them in pairs, storing the results in a new folder. Then, you go through that new folder and combine the images in pairs, storing the results in yet another new folder. You repeat this until you're left with one image.
